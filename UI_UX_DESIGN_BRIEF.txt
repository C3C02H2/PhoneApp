ОБЩО ОПИСАНИЕ
─────────────
DoYouTry е мотивационно мобилно приложение, което задава един прост въпрос
всеки ден: "Did you try today?" Потребителят отговаря с "Yes" или "Not yet",
като записва контекст (какво е опитал, продължителност, настроение) или
причина за пропуск. Приложението следи streak-ове, achievements, седмични
обзори и има social feed за споделяне на истории.

ДИЗАЙН СТИЛ
─────────────
- Dark premium тема
- Минималистичен, без емотикони
- Инспирация: Notion, Linear, Apple Human Interface
- Clean typography, много whitespace
- Меки цветове, без агресивни контрасти

─────────────────
Фонове:
  - Primary:    #0A0A0F   (основен фон, дълбоко тъмно)

================================================================================
                         КОМПОНЕНТИ
================================================================================

StatCard:
  - Карта с secondary фон, subtle border, rounded
  - Цветна лента отгоре (3px, съответния цвят)
  - Голяма стойност (h2)
  - Заглавие (label, muted)
  - Подзаглавие (caption, disabled)
  - flex: 1 (разтяга се в ред)

PrimaryButton:
  - Пълен width, accent фон, бял текст
  - Варианти: default (accent), danger (error)
  - Размери: medium, large
  - Loading state: ActivityIndicator
  - Disabled state: opacity 0.5

TextInputField:
  - Label (muted)
  - Input с secondary фон, subtle border, rounded
  - Error state: error border + error текст
  - Secure text entry support

PostCard:
  - Secondary фон, rounded, subtle border
  - Avatar + author info
  - Category badge + prompt tag
  - Title
  - Footer: like, views, comments
  - Shadow: small

Toggle/Switch:
  - Custom track (44x24, rounded)
  - Thumb (20x20)
  - Active: accent фон + бял thumb
  - Inactive: elevated фон + disabled thumb

Chip:
  - Rounded full, border, tertiary фон
  - Selected: accent glow фон + accent border
  - Text: label size, muted/accent

Section Header:
  - Uppercase, letterSpacing 1.5, disabled цвят
  - Малък шрифт (labelSmall)

Settings Row:
  - Flex row, space-between
  - Label (body, main) + value/arrow/toggle (right)
  - Разделител: subtle border bottom
