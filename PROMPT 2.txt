Искам да създадеш database schema за мотивационно мобилно приложение.

PostgreSQL

Основни функции на приложението:
- регистрация
- login
- потребителски профил
- блог постове
- дневен check-in
- streak tracking

Създай SQL schema за следните таблици:

Users
- id
- username
- email
- hashed_password
- created_at

Posts
- id
- title
- content
- author_id
- created_at

DailyCheckins
- id
- user_id
- date
- answer (yes/no)
- created_at
